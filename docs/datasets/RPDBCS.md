# RPDBCS

## Dataset Description


## Dataset Structure

- 

- Rotation: 37.29 ~ 61.61 Hz

- Classes: Normal, Rubbing, Faulty Sensor, Misalignment and Unbalance

- Domain: Time

- Sample rate:  samples per second
